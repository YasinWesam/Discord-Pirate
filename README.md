### Description:
Discord pirate is a prank discord bot that forces your members to talk like pirates in a specific channel.
You'll have to talk with lowercase, no punctuation and also include a pirate word:

### Installation:
1. Open cmd in the project folder.
2. Type ``` pip install -r requirements.txt ```
3. Set up your channel ID in bot.py.
4. Set up your env file.
5. Run the bat file.
